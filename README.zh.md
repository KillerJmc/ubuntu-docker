# ubuntu-docker

## 链接

+ [转到 Docker Hub](https://hub.docker.com/repository/docker/killerjmc/ubuntu)


## 介绍

一个多功能的Ubuntu docker镜像

## 用法

拉取镜像:

```shell
docker pull killerjmc/ubuntu
```

运行:

```shell
docker run -it killerjmc/ubuntu
```

## 特性

+ 默认添加 **Oh My Zsh**
+ 更换镜像源到 **中科大**
+ 设置默认时区为 **上海**
+ 开箱即用的 **jdk** 和 **mysql**
